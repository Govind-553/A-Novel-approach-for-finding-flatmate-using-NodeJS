# Security Policy

## Supported Versions

Only the latest version of this project (main branch) is supported with security updates.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ |
| Older  | ❌ |

---

## Reporting a Vulnerability

If you find a security issue, please report it responsibly.

- Do **not** open a public issue
- Contact the repository owner via GitHub

### Please include:
- Description of the issue
- Steps to reproduce
- Affected components (if any)

We will review and address valid security issues as soon as possible.

Thank you for helping keep this project secure.
